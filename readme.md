## Структура репозитория

- `01_generation/` — генерация молекул  
  - `input/` — исходные скаффолды и параметры генерации  
  - `output/` — сгенерированные молекулы  

- `02_docking/` — докинг молекул к IRAK4  
  - `input/` — подготовленные молекулы и структура белка  
  - `output/` — результаты докинга  

- `03_selectivity_ml/` — предсказание селективности по панели киназ  
  - `input/` — набор молекул и данные по off-target киназам  
  - `output/` — предсказанные значения селективности  

- `04_admet_rersa/` — фильтры ADMET, drug-likeness и ReRSA score  
  - `input/` — молекулы после докинга и селективности  
  - `output/` — отобранные кандидаты  

- `05_md/` — молекулярная динамика топ-кандидатов  
  - `input/` — комплексы белок-лиганд  
  - `output/` — результаты симуляций (RMSD, H-bonds)  

- `06_retro/` — ретросинтез и оценка синтезируемости  
  - `input/` — финальные молекулы  
  - `output/` — маршруты синтеза и количество шагов  

- `07_overall/` — интеграция результатов всех этапов  
  - `input/` — данные из всех предыдущих этапов  
  - `output/` — итоговый пул кандидатов для экспериментов  

## Установка окружения

1. Перейдите в папку с репозиторием:

```bash
cd irak4
````

2. Создайте окружение из файла `environment.yml` с именем `irak4`:

```bash
conda env create -f environment.yml -n irak4
```

3. Активируйте окружение:

```bash
conda activate irak4
```

4. Проверьте, что все зависимости установлены:

```bash
conda list
```
